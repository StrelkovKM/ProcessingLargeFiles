## Сборка и запуск: 

1. Клонировать репозиторий и перейти в ветку `delevop` или tag: `V2.0.0`.
   Для сборки проекта понадобиться `CMake` версии не ниже `3.10`
2. В директории `ProcessingLargeFiles` открыть консоль и ввести следующие команды:
   1. `mkdir build && cd build`
   2. `cmake ..`

3. В директории `src` в файле `main.cpp` в строку:
   ```cpp
   FileProcessing A("<path_file>"); 
   ```
   вставьте глобальный путь до считываемого файла

   Для `Visual Studio 2022`:
4. В папке build открыть `ProcessingLargeFiles.sln`
5. В проекте назначить исполняемый проект - `src`
6. Запустите программу (F5)

   Для `Visual Studio Code`:
4. Из директории `build` собрать зависимости и обновить `main.cpp` - `make`
5. Запустить исполняемый файл `./src/src`


### Констурктор самостоятельно запрашивает сколько свободной RAM есть у компьютера, если нужно ее задать вручную использовать метод SetRAMSize, аргументом которого является объем памяти в байтах
