## Как запустить:

1. Клонировать ветку develop данного репозитория
2. В папке ProcessingLargeFiles открыть консоль и ввести следующие команды:
   1. `mkdir build`
   2. `cd build`
   3. `cmake ..`
3. В папке build открыть `ProcessingLargeFiles.sln`
4. В проекте назначить исполняемый проект - `src`
5. В `main.cpp` в строку:
   ```cpp
   FileProcessing A("D:/Рабочий стол/ProcessingLargeFiles/src/FileForTest16GB.txt");
вставить путь до считываемого файла и запустить выполнение программы
